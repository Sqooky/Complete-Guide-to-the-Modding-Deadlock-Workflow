## Source 2 Development
These are the tools for actually making content for the game, ie converting your creations to something the game can read.
- [Source2Viewer]() Source2Viewer (or S2V) is used almost everywhere. It allows for decompilation, extraction, and viewing of the assets within deadlock. It is a fantastic tool
- [CSDK12]() The community source development kit for deadlock. It's stitched together from CS2 and Deadlock bits but works incredibly well
### Source 2 Dev Tools For Adding Animation Support
These tools will not be needed unless you are specifically making a character model replacement.
- [Dotryen's Deadlock Tools]() Dotryen's deadlock tools. This tool allows for adding animgraph2 data to models exported from CSDK12. Without this tool models will just A-Pose
- [CSWIN64]() This is all of the CS2 tool binaries made as small as possible so as to be used without installing CS2 proper.
- [CS2 Tools]() Just CS2's tools. Avalible by downloading them from CS2's dlcs
- [Dota 2 Tools]() Just Dota 2's tools. Avalible by downloading them from CS2's dlcs.
